### Выравнивание паддингами флекс элементы
----


```scss
//для того чтобы раздвинуть колонки друг от друга нужно сделать так
.columns__row {
	display: flex;
	flex-wrap: wrap;
	margin: 0 -8px;
}
.columns__column {
	padding 0px 8px;
	flex: 0 1 50%;
}

//таким образом мы получаем отступ друг от друга в 16px
```
